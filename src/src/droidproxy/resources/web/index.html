        Factory integration

        Wire DroidProxy into Factory. The custom models entry edits ~/.factory/settings.json; the Challenger Droids are copied into ~/.factory/droids/ and ~/.factory/commands/.

        After applying, restart Factory (or open a new session) so the model picker picks up the new entries.
